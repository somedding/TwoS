# TwoS

크롬 익스텐션을 사용해서 인터넷 사용 시간 측정하고 공유

현재 구현된 기능
- 크롬 익스텐션으로 크롬 사용시간 측정
- 들어간 도메인 종류 별로 분류
- 인당 총 이용 시간
- 월 단위, 주 단위 전체 평균
- 이시간에 다른 것을 했으면
- community 친구 추가 , 관리

아직 수정해야 하는 기능
- data 파트
- 서버 이용

## 사용방법
1. 크롬을 키고 `chrome://extensions/` 들어가기
2. 개발자 모드 키기
3. 압축파일 해제후 해당 폴더를 `압축해제된 확장 프로그램을 로드합니다` 에 로드
4. 확장프로그램 최초 1회 실행
5. `twos.gay` 들어가서 로그인 후 시간 확인
