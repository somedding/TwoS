# TwoS

크롬 익스텐션을 사용해서 인터넷 사용 시간 측정하고 공유

현재 구현된 기능
- 크롬 익스텐션으로 크롬 사용시간 측정
- 들어간 도메인 종류 별로 분류
- 인당 총 이용 시간
- 월 단위, 주 단위 전체 평균
- 이시간에 다른 것을 했으면
- community 친구 추가 , 관리

아직 수정해야 하는 기능
- data 파트
- 서버 이용
